what's the difference between local storage and session storage ?



local storage :

1.Local storage data is permanent.

2.Data has to be deleted manually.

3.Works for any tab, not for any specific tab.

4.The data is not deleted when the browser is closed, and are available for future


session storage :

1.Session storage data is temporary.

2.Closing the browser or tab clears the storage.

3.Works for a specific tab and when the tab is closed the data is deleted.
